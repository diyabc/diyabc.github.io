Welcome to the webpage of the software DIYABC (Do It Yourself ABC):

a user-friendly approach to Approximate Bayesian Computation for inference on population history using molecular markers


by Jean-Marie Cornuet, Pierre Pudlo, Julien Veyssier, Etienne Loire, Alexandre Dehne Garcia, Filipe Santos and Arnaud Estoup
Centre de Biologie et de Gestion des Populations, INRA, Campus International de Baillarguet, CS30016 34988 Montferrier-sur-Lez, France

Dear DIYABC users,

We have recently made available (July 2015) a new version of the program DIYABC (DIYABC v2.1.0). This new version includes the following major and minor improvements:

Major improvements

(you can download the following file DIYABC_2_1_0_major_improvements.pdf for additional details)
  1. New analysis options to compute error / accuracy indicators conditionally to the observed dataset.
  2. Possibility to specify a MAF (minimum allele frequency) criterion on the analyzed SNP datasets.
  3. Optimization of the simulation process of SNP datasets that include a substantial amount of missing data.

Minor improvements

  1. We have fixed several memory leaks that could be lethal for computers with low memory capacity.
  2. We have improved consistency tests on the observed datafile format (separators, locus-type definition, missing data, monomorphic SNP) with more explicit warning messages.
  3. We have fixed a bug that made it difficult to install the linux version of the program without local compilation of the source files.
  4. We have fixed bugs related to various operating systems encoding formats.
  5. We have corrected minor bugs related to the production of simulated datasets and data analyses.
  6. We have written a small computer program to convert a vcf format SNP datafile (which corresponds to a widely use file format for SNP) into a DIYABC format SNP datafile. The program is a Python 3.x script which can be run under linux and windows operating system. This tool, together with a readme file and example files, is freely available at https://github.com/loire/vcf2DIYABC.snp.

All simulations and analyses that were processed successfully with the previous version of the program do not need to be processed again. Note that a DIYABC projects produced with the previous version of the program might possibly not work with this new version.

Reference of the associated paper : Cornuet J-M, Pudlo P, Veyssier J, Dehne-Garcia A, Gautier M, Leblois R, Marin J-M, Estoup A (2014) DIYABC v2.0: a software to make Approximate Bayesian Computation inferences about population history using Single Nucleotide Polymorphism, DNA sequence and microsatellite data.
